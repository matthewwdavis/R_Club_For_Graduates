# R Club for Graduates

Materials for a graduate-level R club focused on practical data analysis,
reproducible workflows, and hands-on scripting in R.

## Repository Structure

- `R_Club_Scripts/`
	- `2022_08_29_chapter2.Rmd` and `.html`
	- `2022_09_26_chapter4.Rmd` and `.html`
	- `2022_10_05_chapter5.Rmd` and `.html`
	- `predicting_ck_measurement.Rmd`
	- `R_Club_Scripts.Rproj`

## What This Repository Is For

This repository collects lesson scripts and rendered outputs used during the
graduate R club sessions. It is intended as both:

- A teaching resource during meetings
- A reference for participants to review and reuse code patterns

## Getting Started

1. Open `R_Club_Scripts/R_Club_Scripts.Rproj` in RStudio.
2. Open any `.Rmd` file to review lesson notes and code.
3. Knit files to HTML to regenerate outputs if needed.

## Notes

- Existing `.html` files are rendered versions of the corresponding `.Rmd` files.
- Temporary RStudio/session files are excluded with `.gitignore`.
